# Codex Neural: Live Preview Access

## Access Protocol
The platform is currently optimized and accessible via the local development or production server.

**Local Access:**
```bash
npm run dev
# Access at http://localhost:3000
```

**Production Preview:**
```bash
npm run build
npm start
# Access at http://localhost:3000 (Optimized Mode)
```

## Active Optimization Features
- **Predictive UX**: Hover over "Services" cards to see intent tracking.
- **Adaptive Quality**: Resize window or simulate low FPS to see particle density drop.
- **Lazy Loading**: Portfolio images use `via.placeholder.com` for optimized loading.
- **Ambient Logic**: Background effects disable on low-end devices.

## Telemetry Console
Open browser console (F12) to view real-time logs:
- `[Engagement]`: Intensity tracking
- `[Predictive]`: Intent classification
- `[System]`: FPS drops and Quality Mode changes
